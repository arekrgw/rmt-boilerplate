# React-Typescript Boilerplate

## Stack

- React@17
- MobX@6
- Chakra-UI@1.3
- Emotion@11
- React-query@3
- Typescript@4.1
- Webpack


## Development

- eslint
- prettier
- husky
- jest
- @testing-library/react


## To run this project do:

```
nvm use
yarn
yarn start
```
